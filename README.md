# Blueprint — Balatro Wikipedia

Fan-made wiki for the game [Balatro](https://store.steampowered.com/app/2379780/Balatro/), developed by LocalThunk and published by Playstack.

## Features

- Wiki pages for Jokers, Tarot cards, Planet cards, Spectral cards, Vouchers, Blinds, Enhancements, and Editions
- Search and filter by rarity
- Score calculator
- Synergy checker
- Blind calculator
- Achievement tracker
- Spanish and English language support
- Mobile responsive

## Tech Stack

- React 19
- TypeScript
- Vite
- OGL (3D background)

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
